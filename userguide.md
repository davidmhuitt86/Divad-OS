# Welcome to Divad OS

Welcome to Divad OS, the Engineering Operating System developed by Divad Technology Group.

Divad OS is the operational interface for the Engineering Knowledge Engine (EKE). Rather than managing files alone, it manages engineering knowledge through a Universal Object Model, allowing every document, task, decision, architecture phase, and engineering artifact to become connected, searchable, versioned, and traceable.

Every page within Divad OS has a single responsibility, but all pages operate on the same underlying knowledge base.

# Creating a New Object

Objects are the core building block of Divad OS.

To create one:

1. Select **Objects**.
2. Click **New Object**.
3. Choose an Object Type.
4. Enter a title.
5. Add a description.
6. Select an owner.
7. Add tags.
8. Link related objects.
9. Click **Create**.

The object is created in **Draft** status.
